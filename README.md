## Dependencies

* [Node.js LTS version](https://nodejs.org/en/)

### Install Dependencies

* `npm install`

This will install all of the dependencies listed in the package.json file.

### Run Scripts

#### Dev server

* `npm start`

This will run the client dev server at http://localhost:8080.

#### ESLint

* `npm run lint`

#### JSDoc

* `npm run jsdoc`

#### Build

* `npm run build`

#### Unit tests

* `npm run test`
