## Dependencies
* [Node.js LTS version](https://nodejs.org/en/)

### Install Dependencies
* `npm install`  

This will install all of the dependencies listed in the package.json file.

## Run Scripts

### ESLint
* `npm run eslint`  

### JSDoc
* `npm run jsdoc`  

### Build
* `npm run build`  

### Run dev server
* `npm start`  

This will run the client dev server at http://localhost:3000.  

### Run unit tests
* `npm run test`
